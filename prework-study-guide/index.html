Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Most of the contents in the head element will not be visible to the user.
  • Make sure anything you want visible to the user is in the body section.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • You can assign colors using semantic references, but you van also refer to a color using its hexadecimal code.
  • You can group together multiple elements using a comma (,) so it is more efficent and easier to read.
  • You can assign width and height to elements and images.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new brnach and switches to it
  • git commit -m: commit your staged content as a new commit snapshot
  • git pull origin branch-name: fetch and merge any commits from a tracking remote branch
  • git push origin branch-name: transmit local branch commits to the remote repository branch
